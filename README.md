# SALARY-INCREAMENT-CALCULATOR
A beginner Python and NumPy project that calculates employee salary increments by applying percentage-based raises and generating an updated payroll report.   

💼 Employee Salary Increment Calculator using NumPy

A beginner-friendly Python project that calculates updated employee salaries after applying a percentage increment. The program takes employee IDs, job roles, current salaries, and increment percentage as input, then generates a salary update report.

🚀 Features
Input Employee IDs
Input Employee Job Roles
Input Current Salaries
Apply Salary Increment Percentage
Calculate Updated Salaries
Generate Employee Salary Report
Uses NumPy Arrays for efficient calculations

🛠️ Technologies Used
Python 3
NumPy
